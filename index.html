CH 04

The Minimum Viable Version

"Not polished. Not complete. Not impressive. Real and useful."
MVV = the smallest, simplest version of your project that is real enough to exist in the world and useful enough for someone to actually engage with.
WORKSHEET — My Minimum Viable Version
Fill this out for the project you chose in Ch 3. Be ruthlessly specific. If you write "I need a website" — stop and ask if that's essential or an excuse.
My project is:
The minimum version I can launch is:
To launch it I actually need:
List only true necessities — things without which the project literally cannot exist.
Things I thought I needed but actually don't:
Be honest. This list is usually longer than the one above.
I can have this ready by:
Pick a specific date. Not "soon." A date.
Do This Now

Fill out the MVV worksheet. Every field. Specifically.

If it takes longer than 10 minutes, you're overthinking the "things I need" section. Go back and cut it down. The date you write is your target.

"Action reveals purpose. You don't find it by thinking — you find it by doing."
